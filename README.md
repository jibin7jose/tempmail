# TempMailo üìß

TempMailo is a premium, high-performance temporary email service built with **Next.js 15**, **TypeScript**, and **Vanilla CSS**. It provides users with instant, disposable email addresses to protect their privacy and keep their real inboxes clean from spam.

![TempMailo Preview](https://via.placeholder.com/1200x600/050505/6366f1?text=TempMailo+Premium+UI)

## ‚ú® Features

- **Instant Generation**: Get a random email address the second you land on the page.
- **Premium Aesthetics**: Sleek dark mode design with glassmorphism and smooth animations.
- **Real-time Inbox**: Automatic polling every 10 seconds to catch incoming emails instantly.
- **Privacy First**: No personal data collected. No accounts needed. Just browse and receive.
- **Responsive Design**: Works beautifully on mobile, tablet, and desktop.
- **Copy to Clipboard**: One-click copy for quick use in registration forms.
- **HTML/Text View**: Full support for viewing both HTML and plain text emails.

## üöÄ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (Global Styles & CSS Variables)
- **Icons**: [Lucide React](https://lucide.dev/)
- **API**: [Mail.tm](https://mail.tm/)
